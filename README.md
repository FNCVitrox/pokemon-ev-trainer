# Pokemon EV Trainer

Eine mobile-freundliche Webapp zum Planen und Tracken von EV-Training in Pokemon Feuerrot / Blattgruen.

## Features

- Kanto-Pokemon-Auswahl mit Suche
- EV-Ziele und Fortschritt
- Build-Empfehlungen
- Pokérus- und Machoband-Multiplikator
- Undo fuer Trainingsklicks
- Shiny-Anzeige
- Tutorial fuer EV-Training
- Deutsch/Englisch-Schalter

## Nutzung

Die App ist eine statische Webseite. Sie kann direkt ueber `index.html` geöffnet oder per GitHub Pages gehostet werden.
